# Chapter 1 Resources

A growing glossary of references for Python: official sources, learning resources, tools, and community. Entries are added as topics are written.

## Official

- [python.org](https://www.python.org/) - the official home of Python: downloads, documentation, news
- [Python Documentation](https://docs.python.org/3/) - the official docs, including the tutorial, library reference, and language reference
- [PEP Index](https://peps.python.org/) - all Python Enhancement Proposals, the public design documents through which Python evolves
- [PEP 8: Style Guide for Python Code](https://peps.python.org/pep-0008/) - the community style conventions
- [PEP 20: The Zen of Python](https://peps.python.org/pep-0020/) - the nineteen aphorisms behind Python's design
- [Python documentation by version](https://www.python.org/doc/versions/) - every release and its status
- [endoflife.date/python](https://endoflife.date/python) - at-a-glance support status of each Python version

## Learning Resources

- [The Official Python Tutorial](https://docs.python.org/3/tutorial/) - the tutorial maintained alongside the language itself
- [Real Python](https://realpython.com/) - high-quality articles and tutorials across all levels
- [CS50P: Introduction to Programming with Python (Harvard)](https://cs50.harvard.edu/python/) - free university-level introduction
- [Python MOOC (University of Helsinki)](https://programming-25.mooc.fi/) - free, exercise-heavy introduction and advanced course
- *Fluent Python* by Luciano Ramalho - the depth reference for idiomatic and advanced Python

## Community

- [Python Software Foundation](https://www.python.org/psf-landing/) - the nonprofit behind Python
- [discuss.python.org](https://discuss.python.org/) - the official discussion forum where language development happens in the open
- [PyCon](https://pycon.org/) - the main Python conference series worldwide
