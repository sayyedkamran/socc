# SOCC

**SOCC (Syed's Open Source Computer Science Curriculum)** is an open-source effort to build a comprehensive, industry-oriented Computer Science curriculum.

The goal is to provide a structured learning path covering Computer Science from first principles to advanced engineering topics.

## Current Status

🚧 Under active development.

### Available Tracks

- Artificial Intelligence (In Progress)

Additional tracks such as Blockchain, Operating Systems, Distributed Systems, Databases, Networking, and others will be added over time.
